Autonomous Exploration
This application allows micro-UAVs to venture into confined, underground areas without any previous knowledge of the environment. Without any need for external positioning information, like GPS, this system is truly autonomous and can be deployed anywhere.
  • Featured on BBC Earth
    • Languages and Tools
    • C
    • Python (pyqt)
    • Crazyflie
    • FreeRtos
    Design and Decisions

    To make this system autonomous, it is embedded within the firmware of a CrazyFlie quadcopter. When the application starts a memory efficient mapping and frontier detection system are combined to encourage the UAV to explore by travelling to regions of space yet to be explored. The map provides a way to track where the drone has already been and provides paths to new areas yet to be explored. Frontier detection is responsible for discovering regions that the drone must fly to in order to continue exploration.
